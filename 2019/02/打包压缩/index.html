tar <参数> <文件>

  • tar本身不具有压缩功能。他是调用压缩功能实现的
  • tar 包文件的命令通常都是以 .tar 结尾的。生成 tar 包后，就可以用其它的程序来进行压缩

参数

选项 含义
-A 新增压缩文件到已存在的压缩
-B 设置区块 大小
-b 设置区块 数目
-C 解压到 指定目录
-c 建立 新的压缩文件
-f 使用 归档文件或设备
-x 释放压缩文件
-r --append 追加 文件
-v --verbose 显示 操作过程
-d 记录文件的差别
-u 添加改变了和现有的文件到已经存在的压缩文件
-t 显示压缩文件的内容
-z 支持gzip解压文件
-j --bip2 支持bzip2解压文件
-J --xz 支持xz压缩
-Z 支持compress解压文件
-l 文件系统边界设置
-k 保留原有文件不覆盖
-m 保留文件不被覆盖
-W 确认压缩文件的正确性
--delete 从归档文件删除

gzip <参数> <文件>

  • gzip 命令对文本文件有60%～70%的压缩率

参数

选项 含义
-a 使用ASCII文字模式
-c 输出到标准输出设备，不去更动原始文件
-d 解压缩
-f 强行压缩文件
-l 列出压缩文件的相关信息
-L 显示版本与版权信息
-n 压缩文件时，不保存原来的文件名称及时间戳记
-N 压缩文件时，保存原来的文件名称及时间戳记
-q 不显示警告信息
-r 递归处理
-t 测试压缩文件是否正确无误
-v 显示指令执行过程
-num 指定 num 调整压缩的速度。默认 6
-1 --fast 最快压缩方法（低压缩比）
-9 --best 最慢压缩方法（高压缩比）

实例

dd <选项>

选项 含义
if=<文件> 从指定文件读取
of=<文件> 写入到指定文件
ibs=<比特数> 一次读取的比特数，默认 512B
obs=<比特数> 一次写入的比特数，默认 512B

备份系统

说明

  • 获取 root 权限，进入文件系统的根目录
  • cvpfz 创建档案文件、保持权限(保留所有东西原来的权限)、使用 gzip 来减小文件尺寸
  • backup.gz 得到的档案文件的文件名
  • / 要备份的目录，在这里是整个文件系统
  • 在档案文件名 backup.gz 和要备份的目录名 / 之间给出了备份时必须排除在外的目录–exclude=`
  • backup.gz 这个档案文件本身必须排除在外。备份结束时可能会有提示：tar: Error exit delayed from previous errors 多数情况下可以忽略
