查看内核

lsmod

显示内核中模块的状态

lsmod <选项> <模块名>

显示模块信息

选项 含义
-a 显示模块开发人员
-d 显示模块描述
-p 显示模块支持的参数
-l 显示模块版权声明
-n 显示模块文件名

modprobe <选项> <模块名>

选项 含义
-c 显示所有模块设置信息
-d 使用排错模式
-r 卸载模块
-v 显示详细信息

shutdown

选项 含义
-k 不执行关机，发出警告
-r 重新启动
-h 关闭系统
-c 取消关机

halt/reboot

选项 含义
-w 不关闭，写入 /varlog/wtmp 文件
-d 关闭，不写入文件
-f 强制关闭

uname <选项>

选项 含义
-a 显示全部信息
-m 显示计算机硬件结构名称
-n 显示网络主机名称
-r 显示操作系统内核发行号
-s 显示操作系统名称

hostname <名称>

选项 含义
-a 显示主机别名
-i 显示 IP 地址
-d 显示 DNS 域名
-y 显示 NIS/YP 域名
-F<文件> 读取名称

cal <选项>

选项 含义
-j 给定月的每一天是一年的第几天
-y 显示整年的日历
-m 以星期一为每周第一天的方式显示
-s 以星期日为每周第一天的方式显示
-3 显示前一个月，本月，下个月的日历

date <选项> <格式>

选项 含义
-r<文件> 显示文件的最后修改时间

hwclock <选项>

选项 含义
-r 读取并显示硬件时钟
-w 使用当前系统时间调整硬件时间
-u 把硬件时钟设置为 UTC

systemctl <选项> <命令>

系统命令

系统命令 含义
halt 关闭
poweroff 关闭并切断电源
reboot 重启
suspend 暂停

选项

选项 含义
-r 显示主机和本地容器的单元列表
-q 抑制输出
-a 显示所有加载的单元
-H <主机> 在远程主机操作
-t <类型> 列出特定类型单元

服务名令

  • start
  • stop
  • status
  • restart
  • try-restart
  • reload
  • isolate
  • kill
  • is-active
  • is-failed

free <选项>

显示系统使用和空闲的内存情况，包括物理内存、交互区内存 swap 和内核缓冲区内存。共享内存将被忽略

选项

选项 含义
-b 以Byte为单位显示内存使用情况
-k 以KB为单位显示内存使用情况
-m 以MB为单位显示内存使用情况
-g 以GB为单位显示内存使用情况
-o 不显示缓冲区调节列
-s <间隔秒数> 持续观察内存使用状况
-t 显示内存总和列
-V 显示版本信息
