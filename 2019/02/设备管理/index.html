UUID

通用唯一识别码

  • 一组小写 十六进制 数字序列
  • 由 - 分成五组 ，32 个数字，128位
  • 8位数字 + 3组 4位数字 + 12位数字
    • 当前日期和时间
    • 时钟序列
    • 全局唯一 IEEE 机器识别码

查看 UUID

  • /dev/disk/by-uuid 文件
  • blkid 命令

/etc/fstab 文件

  • 设备
  • 挂载点
  • 文件系统类型
  • 挂载选项
    • ro
    • rw
    • sync
    • async
    • defaults
    • owner
  • 转储选项
  • 文件系统检查选项

fdisk <选项> <设备>

参数

选项 含义
-l 列出指定磁盘分区表
-s <分区> 显示分区大小

交互命令

mkfs <选项> <设备>

创建文件系统，也可使用 mkfs.ext4 / mkfs.vfat 等命令

参数

选项 含义
-t 指定文件系统类型
-v 显示详细信息

df <选项> <文件>

显示文件系统信息

参数

选项 含义
-a 显示所有文件系统
-i 显示 inode
-k 以 KB 为单位
-T 显示文件系统类型
-h 以可读的方式输出
-H 以 1000 为单位

mount <选项> <设备> <目录>

挂载目录

参数

选项 含义
-t 指定文件系统类型
-a 挂载所有
-o <挂载选项> 指定挂载选项
-r 以只读方式挂载
-w 以读写方式挂载
-L <Label> 以卷标方式挂载
-U <UUID> 以 UUID 方式挂载

挂载选项

选项 含义
ro 只读
rw 读写

umount <选项> <设备|目录>

卸载目录

参数

选项 含义
-a 卸载所有文件系统
-n 卸载时不写入信息
-f 强制卸载
-r 若卸载失败，尝试以只读方式重新挂载

xfs_repair <选项> <设备>

参数

选项 含义
-f 指定文件
-n 只检查文件系统
-v 输出详细信息
-d 修复危险

逻辑卷

Logical Volume Manager，逻辑卷管理
