ls <参数> <目录>

列出目标目录中所有的子目录和文件

参数

-a 列出目录下的所有文件，包括 隐藏文件
-d 将目录像文件一样显示
-h 以容易理解的格式列出文件大小 (如 24M 2G)
-l 显示文件的权限、所有者、文件大小等详细信息
-i 显示 inode
-F 显示文件类型
-t 以文件修改时间排序，-r从远到近
-X 根据扩展名排序
-H 使用命令列中的符号链接指示的真正目的地
-k 即–block-size=1K,以 k 字节的形式表示文件的大小

stat <参数> <文件|目录>

查看文件信息

参数

参数 含义
-L 跟踪链接
-f 显示状态信息
-t 简明形式

pwd <参数>

显示当前工作目录绝对路径

参数

参数 含义
-L 输出链接路径
-P 输出物理路径

cd <目录>

切换当前目录至dirName

实例

cp <参数> <源> <目的>

将源文件复制至目标文件或将多个源文件复制至目标目录。

参数

选项 含义
-a all
-i 覆盖前询问
-l 链接文件而不复制
-n 不要覆盖已存在的文件
-R 复制目录及目录内的所有项目

rm <参数> <文件>

删除一个目录中的一个或多个文件或目录

参数

-f 忽略不存在的文件，从不给出提示
-i 交互式删除
-r 列出的全部目录和子目录均 递归 删除
-v 详细显示进行的步骤

rmdir <参数> <目录>

该命令从一个目录中删除一个或多个子目录项，删除某目录时也必须具有对父目录的写权限

参数

-p 递归删除目录dirname，当子目录删除后其父目录为空时，也一同被删除
-v 显示指令执行过程

mkdir <参数> <目录>

在指定位置创建以 目录 命名的文件夹或目录

参数

-m 设定权限<模式> (类似 chmod)
-p 可以是一个路径名称，即一次可以建立多个尚不存在的目录
-v 每次创建新目录都

实例

  • 创建权限为777的目录

  • 一个命令创建项目的目录结构

touch <参数> <文件>

更改文档或目录的日期时间，包括存取时间和更改时间

参数

-a 只更改访问时间
-m 只更改修改时间
-c 不建立任何文档
-d 使用字符串
-t <time> 指定时间
-r 把 <file1> 的时间，设成和 <file2> 相同的时间

ln <参数> <文件1> <文件2>

创建链接

参数

参数 含义
-i 交互
-s 创建符号链接
-L 将硬链接创建为符号链接引用
-T 将链接文件当做普通文件

which <可执行文件名称>

在PATH变量指定的路径中，搜索某个 系统命令 的位置，并且返回第一个搜索结果

参数

-n 　指定文件名长度，指定的长度必须大于或等于所有文件中最长的文件名
-p 　与-n参数相同，但此处的包括了文件的路径

find <路径> <参数>

用于在文件树种查找文件，并作出相应的处理

参数

-name 按照文件名查找文件
-perm 按照文件权限来查找文件
-user 按照文件属主来查找文件
-group 按照文件所属的组来查找文件
-nogroup 查找无有效所属组的文件
-nouser 查找无有效所属主的文件
-depth 在查找文件时，首先查找当前目录中的文件，然后再> 在其子目录中查找
-follow 如果find命令遇到符号链接文件，就跟踪至链接所> 指向的文件
-newer file1 ! file2 查找更改时间比文件file1新但比文件> file2旧的文件
-type 查找某一类型的文件
-mtime -n +n 按照文件的更改时间来查找文件

  • -n表示文件更改时间距现在n天以内
  • +n表示文件更改时间距现在n天以前

实例

-amin n 查找系统中最后N分钟访问的文件
-atime n 查找系统中最后n24小时访问的文件
-cmin n 查找系统中最后N分钟被改变文件状态的文件
-ctime n 查找系统中最后n24小时被改变文件状态的文件
-mmin n 查找系统中最后N分钟被改变文件数据的文件
-mtime n 查找系统中最后n*24小时被改变文件数据的文件

whereis <参数> <文件>

参数

选项 含义
-b 只搜索二进制文件
-m 只搜索 man 手册
-u 搜索不常见记录

nl <参数> <文件>

计将输出的文件内容自动的加上行号

参数

-b 指定行号指定的方式
-b a 不论是否为空行，也同样列出行号
-b t 如果有空行，空的那一行不要列出行号(默认值)
-n 列出行号表示的方法
-n ln 行号在屏幕的最左方显示
-n rn 行号在自己栏位的最右方显示，且不加 0
-n rz 行号在自己栏位的最右方显示，且加 0
-w 行号栏位的占用的位数
-p 在逻辑定界符处不重新开始计算

more <参数> <文件>

分页阅读，在启动时就加载 整个文件

参数

+n 从笫n行开始显示
-n 定义屏幕大小为n行
+/pattern 在每个档案显示前搜寻该字串（pattern），然后从该字串前两行之后开始显示
-c 从顶部 清屏，然后显示
-p 通过清除窗口而不是滚屏来对文件进行换页,与-c选项相似
-s 把连续的多个空行显示为一行
-u 把文件内容中的下画线去掉

操作命令

按键 含义
Enter 向下n行，默认1行
Ctrl+F、Space 向下滚动一屏
Ctrl+B 返回上一屏
d 向下移动半页
b 向上移动一页
Ctrl+l 刷新屏幕
= 输出当前行的行号
:f 输出文件名和当前行的行号
!命令 调用Shell，并执行命令
q 退出more

less <参数> <文件>

可以随意浏览文件，在查看之前 不会加载 整个文件

参数

操作命令

全屏导航

ctrl + F    向前移动一屏
ctrl + B    向后移动一屏
ctrl + D    向前移动半屏
ctrl + U    向后移动半屏

单行导航

j   向前移动一行
k   向后移动一行

其它导航

G     移动到最后一行
g     移动到第一行
q/ZZ  退出 less 命令
n/p   浏览多个文件，后/前一个

标记导航

ma  使用 a 标记文本的当前位置
'a  导航到标记 a 处

tail <参数> <文件>

用于显示指定文件末尾内容，不指定文件时，作为输入信息进行处理。常用查看日志文件

tac 反向列示

参数

cat <选项> <文件>

连接文件或标准输入并打印

  • 一次显示整个文件
  • 从键盘创建一个文件:cat > filename
  • 将几个文件合并为一个文件

参数

-n 对输出的所有行 编号
-b 对 非空 输出行编号
-s 连续两行以上的 空白行 代换为一行
-E 在每行结束处显示 $
-T 将跳格字符显示为 ^I

file <参数> <文件>

查询文件类型

参数

参数 含义
-v 显示版本信息
-z 探测压缩后的文件类型
-L 允许符号链接
-b 列出辨识结果，不显示文件名
-i 显示 MIME 类型

grep <选项> <模式> <文件>

选项 含义
-i 忽略大小写
-l 显示匹配的文件名
-L 显示不匹配的文件名
-n 加上匹配字符串所在行号
-v 只显示不包含字符串的行
-e<模式> 用模式匹配
-f<文件> 从文件匹配模式
-r 递归读取
